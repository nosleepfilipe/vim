# nosleepfilipe vim setup

This setup will create a Symlink in your home directory to the `vimrc` inside of this folder `.vim`


# Installing


```
git clone git@github.com:nosleepfilipe/vim.git ~/.vim
cd ~/.vim
./setup
```
